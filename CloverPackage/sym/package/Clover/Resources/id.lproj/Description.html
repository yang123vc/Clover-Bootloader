Clover v2 adalah kompilasi open source dari: Clover, rEFIt, XNU, VirtualBox.

EDK2 revisi terbaru. http://sourceforge.net/projects/cloverefiboot/


Fitur-fitur:

- boot into 10.4 - 10.12 macOS, Windows EFI and Linux EFI

- Untuk boot 10.4 - 10.9 OSX, Windows EFI dan Linux EFI

- Clover mendeteksi perangkat keras secara otomatis dan mengaturnya langsung.

Tapi bisa diubah lewat config.plist

- Dengan Clover, anda dapat melakukan restart ke OS lain dari Startup Disk prefPane

- UUID yang di config.plist akan sama dengan yang di OS

- SMBIOS akan dikoreksi ke standard 2.6

- ACPI akan dikoreksi ke standard 4.0.

DSDT custom akan dijalankan dari partisi boot atau dari folder EFI

- DSDT patching on the fly by fixes and by patterns

- Pengaturan RestartFix dan pembenahan untuk fitur sleep/wake, serta nilai PCIRootUID

- Injeksi kartu grafis untuk ATI, NVidia dan Intel dimana lebih banyak kartu yang didukung.

Yang juga bisa diatur.

- USB fix (LegacyOff, Ownership, Builtin, clock-id) sampai dengan USB 3.0

- Injeksi untuk HDA dan Ethernet yang builtin

- CPU turbo dan dapat menghasilkan P- serta C-states.

- Memuat ACPI tabel (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)

- extra kexts loading

- kernel and kext patching on the fly

- set NVRAM variables for different purpose

- Boot setelah timeout, GUI yang bisa diatur, dan simpan screenshots GUI melalui tombol F10

- Simpan boot.log dalam OS dan preboot.log dari GUI dengan tombol F2
