Clover v2 este un software open-source bazat pe diferite proiecte: Clover, rEFIt, XNU, VirtualBox.

Principalul este ultima revizie EDK2. http://sourceforge.net/projects/cloverefiboot/


Facilitati:

- boot into 10.4 - 10.12 macOS, Windows EFI and Linux EFI

- bootloader-ul sau EFI ofera RuntimeServices pentru sistemul de operare, lucru imposibil pentru bootloader-ele traditionale de tip "legacy"

- Clover ofera detectie hardware automata si setarea implicita a tuturor proprietatilor.

Uilizatorul le poate insa modifica din config.plist

- cu Clover, puteti alege sa initiati alt sistem de operare, folosind Startup Disk prefPane

- UUID-ul folosit in config.plist va fi acelasi si in sistemul de operare

- SMBIOS va fi corectat pentru standardul 2.6

- ACPI va fi corectat pentru standardul 4.0.

Un DSDT modificat va fi incarcat de pe partitia de boot sau din folderul EFI

- DSDT patching on the fly by fixes and by patterns

- RestartFix customizabil, fix-uri pentru sleep/wake si valori PCIRootUID

- Injector de Proprietati Grafice pentru ATI, NVidia si Intel pentru o arie larga de placi video suportate.

De asemenea customizabile.

- solutii USB (LegacyOff, Ownership, Builtin, clock-id) pana la USB 3.0

- injector HDA si injector predefinit pentru Ethernet

- CPU turbo si generator de stadii P- si C-

- incarcarea tabelelor ACPI (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)

- extra kexts loading

- kernel and kext patching on the fly

- set NVRAM variables for different purpose

- initiere implicita dupa timeout, interfata customizabila, si salvarea de screenshot-uri din interfata, apasand F10

- salvarea boot.log si preboot.log in sistemul de operare, din interfata grafica, apasand F2
