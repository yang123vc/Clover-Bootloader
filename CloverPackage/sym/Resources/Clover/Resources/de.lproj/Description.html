Clover v2 Ist Open Source basierend auf mehreren Projekten: Clover, rEFIt, XNU, VirtualBox.

Der Haupbestandteil ist EDK2 in der letzten Revision. http://sourceforge.net/projects/cloverefiboot/


Hauptmerkmale:

- boot into 10.4 - 10.12 macOS, Windows EFI and Linux EFI

- Der EFI Bootloader stellt RuntimeServices für das OS bereit, was mit Legacy Bootloadern nicht möglich ist

- Clover ist größtenteils in der Lage die nötigen Einstellungen passend zur Hardware selbst vornehmen.

Der Benutzer kann dies aber auch mit der config.plist selber einstellen

- Mit Clover kann man direkt über Startvolume in den Systemeinstellungen in ein anderes OS booten

- Die in der config.plist eingestellte UUID wird im OS gleich bleiben

- SMBIOS wird auf Standard 2.6 angepasst

- ACPI wird auf Standard 4.0 korrigiert.

Die eigene DSDT wird von der gebooteten Partition oder vom EFI Ordner geladen

- DSDT patching on the fly by fixes and by patterns

- Anpassbare Korrekturen für den Neustart und Energiesparmodus, sowie den PCIRootUID Wert

- Injektor für Video-Eigenschaften für ATI, NVidia und Intel für eine große Anzahl von unterstützten Karten.

Auch anpassbar.

- USB fix (LegacyOff, Ownership, Builtin, clock-id) bis zu USB 3.0

- HDA Injektor und Ethernet builtin Injektor

- CPU Turbo und P- und C-states Generator

- Laden von ACPI Tabellen (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)

- extra kexts loading

- kernel and kext patching on the fly

- set NVRAM variables for different purpose

- Automatisches Booten nach Zeitüberschreitung, anpassbare Oberfläche, Sichern von Bildschirmfotos mit F10

- Sichern der boot.log im OS und der preboot.log mit F2
