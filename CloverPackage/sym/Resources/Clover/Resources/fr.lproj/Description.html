Clover v2 est un programme open source basé sur différents projets: Clover, rEFIt, XNU, VirtualBox.

Le projet principal est EDK2. La dernière révision est disponible ici: http://sourceforge.net/projects/cloverefiboot/


Caractéristiques principales:

- boot into 10.4 - 10.12 macOS, Windows EFI and Linux EFI

- Clover est un chargeur EFI qui fournit un environnement pour différents systèmes d'exploitation ce qui est impossible avec d'autres chargeurs de démarrage

- Clover détecte automatiquement votre matériel et le configure par défaut.

Libre à vous de modifier les paramètres en éditant le fichier config.plist

- Avec Clover vous pouvez changer d'OS à partir du disque de démarrage dans les préférences système

- UUID écrite dans le fichier config.plist sera le même sous OSX

- SMBIOS sera mise à jour au standard 2.6

- ACPI actualisé au standard 4.0.

Votre propre DSDT peut être pris en charge à partir de la partition active ou depuis le dossier EFI

- DSDT patching on the fly by fixes and by patterns

- RestartFix ,sleep/wake et valeur PCIRootUID personnalisable

- Injecteur Propriétés Graphiques pour ATI, Nvidia et Intel pour un large choix de cartes.

Également paramétrable.

- Corrections USB (LegacyOff, Ownership, Builtin, clock-id) jusqu'à l'USB 3.0

- Injecteur HDA (son) et Ethernet intégré

- Générateur pour CPU turbo, et les états P et C

- Chargement des tables ACPI (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI…)

- extra kexts loading

- kernel and kext patching on the fly

- set NVRAM variables for different purpose

- Démarrage par défaut après timeout, interface graphique modifiable, sauvegarde de capture d'écran par GUI avec F10

- Sauvegarde du fichier boot.log dans l'OS et preboot.log par GUI en tapant F2
